# Formato de archivos

## port[1-5].txt
Problemas de prueba
```
Numero de activos (N)
para cada activo i (i=1,...,N):
     rendimiento promedio, desviacion estandar del rendimiento
para cada posible par de activos:
     i, j, correlacion entre activo (i, j)
```

## portef[1-5].txt
Las fronteras eficientes sin restricciones para cada conjunto de datos
```
Para cada uno de los puntos calculados en la frontera:
     rendimiento promedio, varianza del rendimiento
```
